# PLANIFICADOR DINÁMICO TAREAS (Smart TODO)

## Objetivo del programa:
Esta aplicación permite a un usuario gestionar tareas y agruparlas por categrías personalizadas, permitiendole agregar nuevas categorías, agregar tareas a las categorías, visualizar categorías y tareas, aplicar un filtro para visualizar únicamente una categoría y mostrar un tablero general con los datos. La única regla es que el manejo de tareas es mediante la ultima ingresada, conocido como FIFO (First In, First Out), esto hace que sí o sí realicé las tareas en el orden que se agregaron.
